📘 Student Result Inquiry System | نظام الاستعلام عن نتيجة الطالب

Description | الوصف
A responsive web page to check student grades by national ID. Modern UI with animated background, glass cards, and RTL support.
صفحة ويب تفاعلية لعرض درجات الطلاب باستخدام الرقم القومي، مع تصميم حديث وتأثيرات متحركة ودعم RTL.

Features | المميزات

Select stage & grade dynamically | اختيار المرحلة والصف ديناميكيًا

Validate 14-digit national ID | التحقق من الرقم القومي (14 رقم)

Animated gradient background & floating circles | خلفية متدرجة متحركة ودوائر عائمة

Glassmorphism cards for search & results | بطاقات بزجاجية للشاشة والنتائج

Table with core & non-core subjects, max/min, and total | جدول بالمواد الأساسية وغير الأساسية، الكبرى/الصغرى، والمجموع

Tech | التقنيات

HTML5, CSS3, JavaScript | HTML5، CSS3، JavaScript

Google Fonts (Cairo) | خطوط جوجل (Cairo)

Optional: Google Apps Script backend | اختياري: Backend باستخدام Google Apps Script

Usage | طريقة التشغيل

Open result.html in browser | افتح الملف في المتصفح مباشرة

Or connect to Google Apps Script backend | أو ربطه مع Google Apps Script للبيانات الحقيقية
